## 一、产品定位

### 1.1 定位升级

#### 原定位
> 个性化出海资讯、咨询、服务一站式平台，类似OpenAI Pulse、Perplexity Discover，成为中企出海助手

#### 优化定位
> **中国企业的AI出海决策助手**
>
> 通过AIGC深度加工全球商业情报，结合智能工具与专家服务，帮助决策者洞察全球脉动、抢占出海先机

#### Slogan
```
中文：AI驱动，深度加工，全球脉动
英文：AI-Powered Global Insights for Chinese Businesses Going Global
```

### 1.2 产品愿景

**短期愿景（1年）：**
成为中国出海企业决策者每天必看的AI资讯平台

**中期愿景（3年）：**
成为中国出海领域最具影响力的智能决策服务平台

**长期愿景（5年）：**
成为全球中企出海生态的核心基础设施

### 1.3 目标用户

#### 核心用户画像

**用户1：出海企业CEO/创始人**
```
基本信息：
- 年龄：30-50岁
- 职位：CEO/创始人
- 企业规模：50-500人
- 出海阶段：探索期或成长期

需求痛点：
- 需要快速了解目标市场动态
- 需要数据支撑战略决策
- 时间宝贵，追求高效信息获取
- 需要专家意见和行业洞察

使用场景：
- 早晨通勤时快速浏览晨报
- 决策前查看市场分析
- 会议前了解行业动态
- 睡前深度阅读专题报告
```

**用户2：出海业务负责人**
```
基本信息：
- 年龄：28-45岁
- 职位：国际业务VP/总监
- 汇报对象：CEO/COO
- 出海经验：3-10年

需求痛点：
- 需要持续跟踪目标市场
- 需要竞品动态监测
- 需要政策法规更新
- 需要实用工具支持

使用场景：
- 每日早晨获取行业快讯
- 撰写报告时查找数据
- 制定策略时参考分析
- 使用工具辅助决策
```

**用户3：投资人/分析师**
```
基本信息：
- 年龄：26-40岁
- 职位：投资经理/行业分析师
- 关注领域：出海赛道
- 投资阶段：Pre-A到C轮

需求痛点：
- 需要行业趋势洞察
- 需要投资机会发现
- 需要数据和报告支撑
- 需要创业公司动态

使用场景：
- 每日扫描投资机会
- 研究时查阅深度报告
- 尽调时获取行业数据
- 投后管理时监测市场
```

#### 用户规模预估

```
市场规模：
├─ 中国出海企业：50万+
├─ 决策层用户：100万+
├─ 潜在付费用户：10万+
└─ 目标1年DAU：5-10万
```

### 1.4 价值主张

```
对用户：
├─ 节省时间：AI筛选精华，无需海量阅读
├─ 支撑决策：深度分析+数据，科学决策
├─ 愉悦体验：电子杂志级美学，享受阅读
└─ 全链服务：内容+工具+服务，一站搞定

对行业：
├─ 提升效率：减少信息不对称
├─ 促进出海：降低决策门槛
├─ 连接生态：企业+服务商+专家
└─ 推动创新：AI驱动内容生产
```

## 二、产品调性

### 2.1 品牌调性

#### 核心关键词
```
1. 专业（Professional）
   - 深度分析而非快讯
   - 数据支撑而非猜测
   - 专家洞察而非拼凑

2. 智能（Intelligent）
   - AI深度加工内容
   - 个性化精准推荐
   - 智能问答助手

3. 美学（Aesthetic）
   - 电子杂志级视觉
   - 品牌统一调性
   - 愉悦阅读体验

4. 可信（Trustworthy）
   - 多源信息标注
   - 专家审核背书
   - 数据可追溯

5. 高效（Efficient）
   - 主动推送晨报
   - 快速获取精华
   - 工具一站集成
```

#### 品牌人格
```
如果Discovery是一个人：
- 性别：中性偏男性
- 年龄：35岁
- 职业：资深出海顾问
- 性格：专业、睿智、高效、友善
- 说话风格：简洁、有深度、有温度
```

```
整体风格：现代、专业、简洁

关键元素：
├─ 卡片：圆角8-12px，轻阴影
├─ 图标：线性风格，2px描边
├─ 配图：16:9或1:1，高质量
├─ 留白：充足间距，不拥挤
└─ 层次：清晰的信息层级
```

### 2.3 内容调性

#### 写作风格

```
标题：
- 简洁有力（8-15字）
- 突出关键信息
- 数据+洞察结合
- 避免标题党

示例：
✅ "东南亚电商GMV破千亿美元，印尼市场增速全球第一"
❌ "震惊！东南亚电商竟然..."

正文：
- 结构化清晰（总-分-总）
- 数据可视化呈现
- 专业但不晦涩
- 观点明确有深度

示例：
✅ "根据Statista数据，2025年东南亚电商市场规模达1,234亿美元，同比增长23%。其中印尼以456亿美元领跑，主要受益于..."
❌ "东南亚电商很火，大家都在做..."

语气：
- 客观中立，避免主观臆断
- 专业权威，但不高高在上
- 有洞察力，不浅尝辄止
- 有温度感，不冷冰冰
```

#### 内容原则

```
1. 深度 > 速度
   不追求最快，追求最深

2. 洞察 > 转载
   AI加工生成，非简单复制

3. 数据 > 观点
   数据说话，观点有据

4. 实用 > 热闹
   解决问题，非博眼球

5. 可信 > 新奇
   来源清晰，内容可靠
```

```
1. 出海政策咨询

   用户：阅读文章后点击"深度追问"

   用户："如何使用MCDA热力图？"
```

```
├─ 即时报道                ├─ 深度加工

└─ 首发信息                └─ 二次加工
```

#### 7.2.1 内容定位

```

├─ AI深度分析（非首发新闻）

├─ 专家观点和洞察
├─ 行业趋势预测
└─ 决策建议和指导

❌ 不应做：
├─ 突发新闻首发报道

```

```
   竞品 = 看资讯
   我们 = 看资讯+用工具+找服务

4. 美学而非功能
   竞品 = 能用就行
   我们 = 电子杂志级体验

5. 决策层而非执行层
   竞品 = 面向运营/专员
   我们 = 面向CEO/VP
```

```
4. 运营执行（5%）

   └─ 口碑传播机制
```
