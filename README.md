## 📊 GECOM 方法论

### 双阶段八模块模型

**阶段 0-1：CAPEX（一次性启动成本）**
- **M1: 市场准入** - 公司注册、许可证、法务咨询、税务登记
- **M2: 技术合规** - 产品认证、商标注册、合规检测
- **M3: 供应链搭建** - 仓储押金、设备采购、初始库存、系统搭建

**阶段 1-N：OPEX（单位运营成本）**
- **M4: 货物税费** - COGS、进口关税、增值税、消费税
- **M5: 物流配送** - 国际运输、本地配送、FBA费用、仓储
- **M6: 营销获客** - CAC、平台佣金、广告投放
- **M7: 支付手续费** - 网关费用（2.9%）、汇率损失（1.5%）、拒付
- **M8: 运营管理** - 客服、人员、软件订阅

### 五步 SOP 流程

1. **战略对齐** - 定义业务目标和目标市场配置
2. **数据采集** - 收集产品信息、定价和业务假设
3. **成本建模** - 计算完整的 CAPEX + OPEX 拆解
4. **场景分析** - 对比不同市场/渠道/策略
5. **洞察与路线图** - 生成 AI 推荐和 90 天行动计划

**解决方案**：通过场景对比功能，分析两种渠道的成本差异和ROI

### 场景3：产品定价
**问题**：不知道如何定价才能保证盈利
**解决方案**：系统自动计算盈亏平衡点，建议最优售价区间

### 场景4：融资路演
**问题**：需要向投资人展示清晰的成本模型
**解决方案**：导出完整的成本分析报告，包含ROI、回本周期等关键指标
