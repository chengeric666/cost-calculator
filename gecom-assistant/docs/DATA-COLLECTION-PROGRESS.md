## 🎯 数据采集方法论

### 数据源分级体系
- **Tier 1 (官方 100%置信度)**: 政府海关、税务局、Amazon官方、物流公司实际报价
- **Tier 2 (权威 90%置信度)**: 行业协会、咨询公司、认证机构
- **Tier 3 (估算 80%置信度)**: AI调研、专家访谈、行业平均值

### 已验证数据源
#### Tier 1官方数据源
- **关税**: USITC (美国)、EU TARIC (欧盟)、Japan Customs、HMRC (英国)、越南海关总署
- **VAT**: 各国财政部/税务局官网
- **物流**: 上海威万国际物流实际报价 (2025-10-30)
- **FBA费用**: Amazon.com/.de/.co.uk/.co.jp官方费率表
- **平台佣金**: Amazon Seller Central、Shopee官方
- **支付费率**: Stripe、PayPal官方费率表

#### Tier 2权威数据源
- **市场准入费用**: 咨询公司报价、行业协会
- **G&A成本**: 行业调研平均值
- **营销费用**: 平台广告数据、ACOS/ACOAS行业标准
